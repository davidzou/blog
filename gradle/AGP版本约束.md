### 插件版本配置位于build.gradle中

```
classpath 'com.android.tools.build:gradle:4.2.2'
```

### Gradle版本配置位于gralde/wrapper/gradle-wrapper.properties中

```
distributionUrl=https\://services.gradle.org/distributions/gradle-6.7.1-all.zip
```
